# Sky-Sprinter- Flappy Astro Game
Develop fun and creative game using html , css and Js


A simple Flappy Bird-style game built using HTML, CSS, and JavaScript.
Control the astro, avoid the pipes, and try to score as high as possible!

✨ Features
🕹️ Smooth gameplay using requestAnimationFrame
🚀 Gravity and jump mechanics
🧱 Dynamic pipe generation
💥 Collision detection
📊 Live score tracking
🔁 Restart functionality

How It Works
* The game uses requestAnimationFrame for smooth animation loops.
* Gravity continuously pulls the astro downward.
* User input applies upward velocity (jump).
* Pipes are generated at intervals with random heights.
* Collision detection checks overlap between astro and pipes.
* Score increases when passing through pipes successfully.
